COMP 3610 Assignment 2 README

This project builds and trains machine learning models to predict NYC Yellow Taxi tip amounts using a data set from January 2024

Setup Instructions

1. Clone the Repository
```git clone https://github.com/<your-username>/comp3610-assignment2.git```
```cd comp3610-assignment2```

2. Install dependancies 
```pip install -r requirements.txt```

3. Run the notebook using cleaned dataset
